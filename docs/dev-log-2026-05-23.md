# BlogPlatform 开发日志 — 2026-05-23

## 移动端响应式适配

### NavBar 汉堡菜单
- 手机端只保留 Logo、搜索框、用户头像在导航栏
- 其余导航项（模板、写文章、部署、消息、退出/登录注册）收入右侧滑出汉堡菜单
- 桌面端布局不变
- 路由切换时自动关闭菜单，body 滚动锁定

### 搜索框聚焦动画
- 点击搜索框时，右侧按钮（头像/登录、汉堡图标）平滑淡出收起
- 搜索框自动扩展占满可用空间
- 失焦后 300ms 过渡恢复原样
- 聚焦时 placeholder 缩短为"搜索..."，留更多输入空间

### 消息页适配
- 手机端：聊天列表和对话界面切换展示（非并排）
- 对话页顶部增加返回按钮（ArrowLeft 图标）
- 桌面端保持左右分栏布局

### 文章卡片布局
- 手机端 `BlogPostCard` 图片移到文字上方（`flex-col-reverse`）
- 图片全宽 + `aspect-[16/9]` 比例
- 用户个人页文章卡片增加 16px 侧边距
- 首页分页按钮、部署页按钮添加 `flex-wrap` 防溢出

### 首页滑动切换
- "发现博客"和"我的关注"标签支持左右滑动手势切换
- 阈值 50px，自动判断横向/纵向避免误触

### 字体响应式缩放
| 页面 | 改动 |
|------|------|
| 文章详情标题 | `text-4xl` → `text-2xl sm:text-3xl md:text-4xl` |
| 博客头名称 | `text-4xl` → `text-3xl sm:text-4xl` |
| 模板页标题 | `text-3xl` → `text-2xl sm:text-3xl` |
| 登录/注册 Logo | 手机端缩小字号和边距 |

### 未登录移动端
- 头像位置显示"登录"按钮
- 注册页邮箱+发送验证码按钮在窄屏纵向排列

## 自定义域名配置

- 购买域名：`ourblog-platform.cn`（腾讯云）
- 平台站 DNS：A `@` → `76.76.21.21` / CNAME `www` → `cname.vercel-dns.com`
- 模板站 DNS：CNAME `blog` → `cname.vercel-dns.com`（子域名，免费）
- 更新 `NEXTAUTH_URL` 环境变量为新域名
- Vercel 自动签发 SSL 证书
